## Data Types & Variables

**Focus:** The fundamental building blocks — how Python stores and manipulates data.

### Topics
- `int`, `float`, `str`, `bool` — the four basic types
- Variables and assignment — naming and binding values
- Type conversion — `int()`, `str()`, `float()`, `bool()`

**Run the demo:**
```bash
python examples/data_types_demo.py
```

---

## Control Flow

**Focus:** Making decisions and repeating actions — the logic of your program.

### Topics
- `if` / `elif` / `else` — branching based on conditions
- `for` loops — iterating over sequences
- `while` loops — repeating while a condition holds
- `break` and `continue` — controlling loop execution

---

## Functions

**Focus:** Organizing code into reusable, named blocks.

### Topics
- `def` and `return` — defining and returning from functions
- Parameters and arguments — passing data into functions
- Default parameters — optional arguments with sensible defaults
- Docstrings — documenting what your function does
