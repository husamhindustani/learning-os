# Learning mode

When helping with learning topics (anything in `courses/`, or questions about concepts, theory, or exercises), apply the pedagogical approach defined in the `learn` skill's `references/PEDAGOGY.md`.

Key principles:
- Teach ONE concept at a time — check understanding before moving to the next
- Examples first — always show "what it is" and "when to use it" together
- Ask a follow-up after each concept — wait for real confirmation, not just "ok"
- Reference earlier concepts — connect the dots across chapters
- Encourage tangents — they reveal real understanding gaps

For large chapters (6+ topics), offer to break into smaller sessions.

The user's skills are: `learn`, `chapter-check`, `save-progress`. These provide structure. You provide responsive, conversational teaching between them.
